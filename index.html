Exercises

Let's git our hands dirty

Install

http://git-scm.com/downloads

Configure


      $ git config --global user.name "Your name"
      $ git config --global user.email "email@example.com"

      $ git config --global core.editor "/usr/bin/vim"

Create a test project


      $ cd /my/working/directory
      $ mkdir git-workshop
      $ cd git-workshop
      $ vim hello.php
          

      <?php
        echo "Hello world!\n";
      ?>

Initialize repository


      $ git init
      Initialized empty Git repository in /my/path/.git/

Add and commit file


      $ git add hello.php
          

      $ git commit -m "Initial commit."
          

      [master (root-commit) 1f73498] Initial commit.
       1 file changed, 3 insertions(+)
       create mode 100644 hello.php

Check status


      $ git status
      On branch master
      nothing to commit, working directory clean

Change the code


      <?php
        echo "Hello $argv[1]!\n";
      ?>
          

      $ php hello.php Pieter
      Hello Pieter!

Check status


      $ git status
      On branch master
      Changes not staged for commit:
        (use "git add ..." to update what will be committed)
        (use "git checkout -- ..." to discard changes in working directory)

        modified:   hello.php

      no changes added to commit (use "git add" and/or "git commit -a")

Check changes


      $ git diff
      diff --git a/hello.php b/hello.php
      index b250c0e..ab9a5dd 100644
      --- a/hello.php
      +++ b/hello.php
      @@ -1,3 +1,3 @@
       <?php
      -  echo "Hello world!\n";
      +  echo "Hello $argv[1]!\n";
       ?>

Add changes to index


      $ git add hello.php

Check status


      $ git status
      On branch master
      Changes to be committed:
        (use "git reset HEAD ..." to unstage)

        modified:   hello.php

Check changes


      $ git diff

Nothing is shown, all changes are added to the index

Check changes ready to be committed


      $ git diff --staged
      diff --git a/hello.php b/hello.php
      index b250c0e..ab9a5dd 100644
      --- a/hello.php
      +++ b/hello.php
      @@ -1,3 +1,3 @@
       <?php
      -  echo "Hello world!\n";
      +  echo "Hello $argv[1]!\n";
       ?>

Commit staged changes


      $ git commit

If the '-m' option is omitted you can enter the log message in an editor.


      Allow to pass user name as an argument.

      # Please enter the commit message for your changes. Lines starting
      # with '#' will be ignored, and an empty message aborts the commit.
      # On branch master
      # Changes to be committed:
      # modified:   hello.php

Output


      [master 0c6bbfb] Allow to pass user name as an argument.
       1 file changed, 1 insertion(+), 1 deletion(-)

Make 2 changes


      <?php
        $name = !empty($argv[1]) ? $argv[1] : 'world';
        echo "Hello $name!\n";
      ?>
          

      $ git add hello.php

Make a second change


      <?php
        // Use 'world' as default name.
        $name = !empty($argv[1]) ? $argv[1] : 'world';
        echo "Hello $name!\n";
      ?>

Check status


    $ git status
    On branch master
    Changes to be committed:
      (use "git reset HEAD ..." to unstage)

      modified:   hello.php

    Changes not staged for commit:
      (use "git add ..." to update what will be committed)
      (use "git checkout -- ..." to discard changes in working directory)

      modified:   hello.php

Check changes to working directory


      $ git diff
      diff --git a/hello.php b/hello.php
      index 23fd8b4..0dc8ed4 100644
      --- a/hello.php
      +++ b/hello.php
      @@ -1,4 +1,5 @@
       <?php
      +  // Use 'world' as default name.
         $name = !empty($argv[1]) ? $argv[1] : 'world';
         echo "Hello $name!\n";
       ?>

Check changes ready to be committed


      $ git diff --staged
      diff --git a/hello.php b/hello.php
      index ab9a5dd..23fd8b4 100644
      --- a/hello.php
      +++ b/hello.php
      @@ -1,3 +1,4 @@
       <?php
      -  echo "Hello $argv[1]!\n";
      +  $name = !empty($argv[1]) ? $argv[1] : 'world';
      +  echo "Hello $name!\n";
       ?>

Commit both change sets


      $ git commit -m "Use 'world' as default user name."

      $ git add hello.php
      $ git commit -m "Add documentation."
